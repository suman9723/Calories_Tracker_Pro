# Calories_Tracker_Pro
The Calorie Tracker is an AI-powered web application that analyzes food images to provide nutritional information. Users can upload or capture food photos, and the system identifies food items and estimates calories, protein, carbohydrates, and fat content.

Calorie_Tracker_Pro_ Web App  
                               A sleek and responsive web application that allows users to upload a food image  
           – Analyzes the contents using AI-powered APIs, and returns nutritional information like calories, protein, carbohydrates, and fats 
           – Uploads image to ImgBB (free image hosting API) .–  
       Analyzes the image using Groq (LLaMA-4) API with multi-modal vision capabilities. 
            – Technologies: HTML5, CSS3, Javascript, ImgBB(API for image hosting),Groq AI API (via Meta LLaMA-4) for food recognition 
and nutrition analysis,Netlify.
